What is Css?

  1. CSS stands for Cascading Style Sheets.
  2. CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
  3. Css is not neccessaryCSS saves a lot of work. It can control layout of multiple webpages at once.
  4. External stylesheets are stored in CSS files.

CSS can be added into HTML document in three ways:

  • Inline - by using the style attribute inside the HTML element.
  • Internal - by using style element in the head section.
  • External - by using the link element to link an external CSS file.
  • The most common way of adding the CSS is to keep the styles in the external CSS files. However, in this tutorial we will use inline and internal styles. because this is easier to demosntrate, and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that css class applies a style to multiple elements. ID on the ither hand, applies a style to one uniquw element. ID is also special in that you can use a special URL to link directly to an element and its used by JavaScript.

In Css selectors are used to target a specific element or range of element on a web page. Once the element is been targeted, a style or set of styles can be applied to the element.

There is a wide range of selectors available. Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
